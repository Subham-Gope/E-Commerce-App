# 🛍️ Forever E-commerce  

Forever E-commerce is a modern and responsive e-commerce platform built with **React**, **Node.js**, **Express**, and **MongoDB**. It offers a smooth user experience with powerful features, secure architecture, and a seamless shopping experience.

## ✨ Features  
- **Fast & Responsive** – Built with React and Tailwind CSS for high performance and a mobile-friendly design.  
- **User Authentication** – Secure login and registration with JWT-based authentication.  
- **Product Management** – Admins can easily add, edit, or delete products from the platform.  
- **Cart Management** – Users can add, remove, and update items in their cart.  
- **Order Processing** – Users can place orders, and admins can manage them.  
- **Real-Time Notifications** – Interactive notifications powered by React Toastify.  
- **Backend Security** – Express with CORS and Dotenv for a secure server environment.  
- **Email Support** – Nodemailer for sending order confirmation emails to users.

## 🛠 Tech Stack  

### Frontend:  
- **React**  
- **Tailwind CSS**  
- **React Hook Form & Zod** (for form validation)  
- **Axios** (for API calls)  
- **React Toastify** (for notifications)  

### Backend:  
- **Node.js**  
- **Express.js**  
- **MongoDB** (for data storage)  
- **JWT** (for user authentication)  
- **Dotenv** (for environment variables)  
- **Nodemailer** (for email handling)  
- **CORS** (for handling cross-origin requests)
